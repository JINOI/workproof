[PRD] SafeBridge: 외국인 근로자 안전 교육 증빙 솔루션
1. 프로젝트 개요 및 타깃 (Step 1)
1.1 제품의 한 줄 정의
안전 관리 가이드를 AI로 요약 및 다국어화하여 외국인 근로자의 교육 이해도를 퀴즈로 검증하고, 법적/운영적 교육 증빙 리포트를 생성하는 안전 온보딩 도구입니다.
1.2 해결하려는 문제
언어 장벽: 한국어 안전 관리 가이드를 이해하지 못하는 외국인 근로자의 사고 위험.
형식적 교육: 단순히 문서를 보여주는 것만으로는 실제 이해도를 확인할 수 없음.
증빙의 어려움: 누가, 언제, 어떤 내용을 얼마나 이해했는지에 대한 데이터 기반 기록 부재.
1.3 핵심 타깃 (Core Target)
관리자: 제조/건설 현장의 안전관리자, 공장장, 노무 담당자.
근로자: 중국, 베트남, 태국 등 한국어 소통이 서툰 외국인 현장 근로자.
2. 누가 무엇을 할 수 있는가? (Step 2: 권한/유저)
2.1 관리자 (Admin/Manager)
조직 관리: 회사/팀 단위로 데이터 격리 관리.
콘텐츠 관리: 안전 관리 가이드 업로드, AI 요약본 수정, 퀴즈 세트 생성 및 관리.
현황 모니터링: 전체 근로자의 교육 이수 현황, 시도 횟수, 리스크 상태 확인.
데이터 추출: 교육 완료 증빙 PDF/CSV 리포트 다운로드.
2.2 근로자 (End User/Worker)
접근: 별도 앱 설치 없이 QR 코드 스캔을 통한 웹 접속.
인증: 간단한 본인 식별 정보(이름, 생년월일) 입력 후 교육 시작.
학습/평가: 모국어 교육 카드 학습 및 100% 정답 기준의 퀴즈 응시.
3. 어떤 흐름으로 움직이는가? (Step 3: 프로세스)
3.1 주요 기능별 프로세스
안전 관리 가이드 등록 or 산업안전보건공단의 표준 안전 관리 가이드 선택: 관리자가 PDF/문서 업로드 → AI가 [작업단계/위험요소/보호구/금지행동] 추출.
다국어 퀴즈 생성: 선택된 언어(중/베/태)(복수선택 가능)별로 AI가 10문항 생성 (O/X, 객관식).
현장 교육: 근로자가 현장에 부착된 QR 스캔 → 모국어 교육 카드 확인(최소 5초 체류) → 퀴즈 응시.
검증 로직: 오답 시 즉시 탈락 및 처음부터 다시 풀기 (문제/보기 순서 랜덤화).
증빙 완료: 전 문항 정답 시 완료 처리 → 관리자 대시보드 실시간 반영.
3.2 데이터의 상태 변화
Pending: 교육 전 또는 퀴즈 미통과 상태.
Safe (안전): 3회 시도 내에 100% 정답 통과.
Warning (경고): 4회 이상 시도 또는 반복적 오답 발생 후 겨우 통과.
3.3 특수 비즈니스 룰
재시도 정책: 한 문제라도 틀리면 "틀린 문제"만 푸는 것이 아니라 전체 세트를 다시 풀어야 함 (학습 효과 극대화).
랜덤화: 재시도 시 문제 번호와 보기 순서가 무작위로 섞임.
시간 지연: 교육 카드를 읽지 않고 넘기는 것을 방지하기 위해 '다음' 버튼 활성화에 5초 딜레이 부여.
4. 기술적인 약속 (Step 4: 개발 스펙)
4.1 기술 스택
Framework: Next.js (App Router) - 빠른 SEO 및 웹 접근성 확보.
Styling: Tailwind CSS - 반응형 모바일 UI 최적화.
AI 엔진: Gemini 2.5 Flash - 안전 관리 가이드 요약, 다국어 번역 및 퀴즈 생성.
Backend/DB: Supabase (PostgreSQL, Auth, Storage) - 실시간 현황판 및 데이터 저장.
Language Support: i18next 또는 별도 AI 번역 레이어.
4.2 주요 데이터 스키마 (Core Table)
Safety_Guide_Master: 제목, 원본 URL, AI 요약(JSON), 생성일.
Education_Log: 근로자ID, Safety_Guide_ID, 시도횟수, 소요시간, 최종점수, 상태(Safe/Warning), 완료타임스탬프.
Quiz_Set: Safety_Guide_ID별 문제, 보기, 정답 데이터 (다국어 포함).

5. 웹페이지 구성
관리자 워크플로우:
랜딩페이지/로그인 창
로그인 하면 관리자 대시보드 페이지-> 상단에 검색창(안전 관리 가이드 검색창) / 왼쪽에 LNB
대시보드에서 과거의 교육 진행했던 안전 관리 가이드 선택하면 과거 내역 페이지로/”새로운 안전 관리 가이드” 선택하면 안전 관리 가이드 업로드 팝업으로
과거 내역 페이지에서는 안전/위험 인원 구분해서 표시하고 각 인원 클릭하면 팝업창으로 타임스탬프/어떤 문제 틀렸는지/몇회 시도 하였는지 확인 가능하도록.
안전 관리 가이드 업로드 팝업에서는 관리자가 문서 업로드하면 AI가 안전 관리 가이드 요약&퀴즈 생성&해당 내용 담긴 QR코드 생성. QR코드 공유하기 버튼도 나와야함




근로자 워크플로우: (QR 들어가면 보이는 창, 듀오링고 디자인 참고)
1. 본인 정보 입력 창 (이름/생년월일)
2. 교육자료 창. AI가 안전 관리 가이드를 기반으로 만들어준 교육 요약본
3.  AI가 안전 관리 가이드를 기반으로 만들어준 퀴즈 10개
4. 만 점이 아니면 재시험
5. 다 맞으면, 걸린 시간, 횟수, "수고하셨습니다!" 창

6. Non-Goals (MVP 범위 제외)
실시간 현장 CCTV 연동.
ERP/MES 시스템과의 데이터 연동.
유료 결제 모듈 (MVP 단계에서는 수동 계약 기반 운영).
10개 이상의 광범위한 다국어 지원 (초기 3개국어 집중).
